## flex

| flex         | display         | "flex","inline-flex"                                                      |
| direction    | flex-direction  | "row","row-reverse","column","column-reverse"                             |
| wrap         | flex-wrap       | "nowrap", "wrap", "wrap-reverse"                                          |
